Glossary : Model

Term - Sistema Telefoni Camere link

Properties
Name Value
Documentation

Sistema esterno che ha la responsabilità di conoscere tutte le chiamate effettuate da tutti i telefoni presenti in ogni camera dell'albergo. Ci si rivolge ad esso per conoscere informazioni in merito alle telefonate emesse da una camera, in un certo periodo.
