Glossary : Model

Term - Stato Camera link

Properties
Name Value
Documentation

Oggetto che ha la responsabilità di conoscere lo stato di una camera al variare del tempo. Per stato si intendono tutte le informazioni dipendenti dal tempo relative ad una camera, se è libera o se è occupata. Se è occupata sarà associata con degli ospiti, ma anche con dei servizi interni con un conseguente prezzo totale di quest'ultimi.
