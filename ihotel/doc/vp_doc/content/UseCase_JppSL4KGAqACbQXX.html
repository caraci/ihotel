Use Case - Chiusura contabile del giorno link

Properties
Name Value
Documentation

Il portiere di notte utilizza il sistema iHotel per effettuare la chiusura contabile del giorno. Il sistema mostra eventuali anomalie nella documentazione generata nella giornata appena finita. Il portiere di notte risolve le anomalie sui documenti. Il sistema mostra l'intera documentazione relativa alla giornata precedente.
